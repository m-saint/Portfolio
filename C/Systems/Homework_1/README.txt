calendar.c
Myles St. Jean
1/19/2023

- displays twelve-month calendar for a particular input year

details:
- prompts user for input year
- finds starting day for the year
- FOR EACH MONTH, prints the month name + the year, the days of the week, and the formatted dates for that month
- accounts for leap years

To compile:
gcc -Wall calendar.c

To run:
./a.exe
